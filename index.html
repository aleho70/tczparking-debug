TCZ Parking

Aplikace pro zobrazení stavu obsazenosti parkovacích míst v budovách Telefonica Praha.

Copyright © 2012 Aleš Holubec

Verze 1.0.0

4.1.2013
  • Základní verze určená k otestovaní širší skupinou uživatelů
  • Testováno na: Galaxy Nexus
  • Podporované platformy:
    • Android: ensure your Android device can install apk files from unknown sources enter Settings --> Applications --> and enable Unkown Sources
    • webOS: You cannot install webOS packages (ipk files) directly from the web; use Palm's palm-install utility for this
    • Symbian: Download and open the wgz file on your device. Done!
    • Windows Phone: You cannot install Windows Phone packages directly from the web; you will need to use Microsoft's tools
  • Nepodporované platformy: iOS, BlackBerry

Verze 0.9.0

3.1.2013
  • Testovací verze pro nové funkce
  • Pro urychlení distribuce nových verzí aplikace s případnými opravami se používá funkce Hydration z cloudového Adobe Phonegap Build. Aplikace proto oproti finální verzi startuje déle a je podepsána pouze development certifikátem.
